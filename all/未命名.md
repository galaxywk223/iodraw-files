autonumber

actor "平台用户" as User
participant "Vue 3 前端" as Frontend
participant "Caddy\n访问入口" as Caddy
participant "Spring Boot\n业务后端" as Backend
database "MySQL" as MySQL
collections "Redis\n缓存" as Redis
queue "Redis Stream\n分析事件流" as Stream
participant "分析消费者\nSpring Boot" as Worker
participant "FastAPI\n模型服务" as Model

== 用户登录 ==

User -> Frontend: 输入账号和密码
Frontend -> Caddy: POST /api/v1/auth/login
Caddy -> Backend: 转发登录请求
Backend -> MySQL: 查询账号、角色和状态
MySQL --> Backend: 返回用户信息

alt 身份验证成功
  Backend --> Caddy: 返回 JWT 和用户角色
  Caddy --> Frontend: 200 OK
  Frontend --> User: 进入角色对应页面
else 身份验证失败
  Backend --> Caddy: 返回认证失败信息
  Caddy --> Frontend: 401 Unauthorized
  Frontend --> User: 显示登录失败提示
end

== 常规同步请求 ==

User -> Frontend: 查询课程或学习数据
Frontend -> Caddy: REST 请求\n携带 Bearer JWT
Caddy -> Backend: 转发业务请求
Backend -> Backend: 身份认证、角色校验\n及数据范围校验

alt 请求合法且具有访问权限
  Backend -> Redis: 查询缓存数据
  alt 缓存命中
    Redis --> Backend: 返回缓存结果
  else 缓存未命中
    Backend -> MySQL: 查询业务数据
    MySQL --> Backend: 返回查询结果
    Backend -> Redis: 更新缓存
  end
  Backend --> Caddy: 返回 JSON 数据
  Caddy --> Frontend: 200 OK
  Frontend --> User: 更新页面内容
else 参数错误或权限不足
  Backend --> Caddy: 返回错误状态和信息
  Caddy --> Frontend: 400 / 401 / 403
  Frontend --> User: 显示错误提示
end

== 学生作答与异步分析 ==

User -> Frontend: 提交题目作答
Frontend -> Caddy: POST /api/v1/courses/{courseId}/answers
Caddy -> Backend: 转发作答请求
Backend -> Backend: 校验身份、课程权限和作答参数

Backend -> MySQL: 开启事务
Backend -> MySQL: 写入作答事件、分析任务\n和待发送事件
Backend -> MySQL: 提交事务
Backend --> Caddy: 202 Accepted + jobId
Caddy --> Frontend: 返回分析任务标识

Frontend -> Caddy: 建立任务状态 SSE 连接
Caddy -> Backend: 订阅分析任务事件

Backend -> MySQL: 读取待发送事件
Backend -> Stream: 发布分析事件
Stream -> Worker: 消费分析任务
Worker -> Model: HTTP/JSON 模型推理请求

alt 模型服务正常
  Model --> Worker: 返回掌握度、表现预测\n和风险预测结果
else 模型服务异常
  Worker -> Worker: 执行规则降级算法
end

Worker -> MySQL: 保存学习画像、学习计划\n并更新任务状态
Worker -> Redis: 更新画像与课程统计缓存
Worker --> Backend: 通知任务处理完成
Backend --> Caddy: SSE 完成事件
Caddy --> Frontend: 推送任务状态

Frontend -> Caddy: 重新请求最新学情数据
Caddy -> Backend: 转发查询请求
Backend -> MySQL: 查询最新画像和学习计划
MySQL --> Backend: 返回最新结果
Backend --> Caddy: 返回 JSON 数据
Caddy --> Frontend: 更新后的学情数据
Frontend --> User: 刷新学习画像和学习计划

@enduml
